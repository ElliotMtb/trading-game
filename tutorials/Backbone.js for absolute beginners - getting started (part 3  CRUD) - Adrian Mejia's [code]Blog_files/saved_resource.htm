    Adrian, excellent tutorial.

    Very small bug, when you update a to-do and enter an empty string, the new empty string will not be saved, which is good, but if you'll double-click again on the same to-do, the input now will be empty instead of containing the to-do text.

    There is more than one way to fix it. I modified the close function to be:

    close: function(){
    var value = this.input.val().trim();
    if(value) {
    this.model.save({title: value});
    }
    else{
    this.input.val(this.model.get('title'));
    }
    this.$el.removeClass('editing');
    },
